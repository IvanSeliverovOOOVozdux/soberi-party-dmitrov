# Soberi Party — Дмитров

Лендинг творческой студии Soberi Party: девичники и мастер-классы для девушек в Дмитрове (Профессиональная 4, офис 427).

Чистый HTML/CSS/JS, без сборки. Деплой статикой на Vercel.

## Структура

- `index.html` — единственная страница лендинга
- `brand/` — лого и шрифты бренда
- `mask-logo.js`, `split-logo.js`, `trace-logo.js` — разовые node-скрипты, которыми готовили SVG-маски логотипа (для прода не нужны)

## Локально

Открыть `index.html` напрямую в браузере — этого достаточно.

## Деплой

Push в `main` → автодеплой на Vercel.
